# DBMS

데이터베이스를 관리하고 운영하는 프로그램(MySQL, 오라클 등등)

대량의 데이터를 효율적으로 관리할 수 있다.

## 관계형 DBMS

관계형 DBMS(RDBMS)는 열과 행으로 이뤄진 테이블로 구성되며, 대부분의 DBMS가 RDBMS 형태로 사용된다.
